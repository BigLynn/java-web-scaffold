# Java web 项目脚手架
基于SpringBoot的rest web基础脚手架，实现java工程快速开发。适用于各种外包项目，拿起就用

## Feature
1. Rest web框架
2. 多模块解耦(api,service,common,consumer等)
3. Mysql 配置
4. Mybatis框架
5. lombok
6. Log输出标准化（LogUtil）
7. Spring Cached with redis
8. Mybatis代码生成
9. model标准化（AbstractEntity）
10. jackJson序列化
11. Mockito 测试框架
12. Actuator 监控
13. 统一Response格式
